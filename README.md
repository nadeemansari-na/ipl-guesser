# 🏏 IPL Player Guesser AI

An AI-powered IPL Player Guessing Game inspired by Akinator, where the system intelligently predicts the IPL player a user is thinking of by asking smart yes/no questions.

Built using probability-based reasoning, dynamic filtering, and Gemini AI integration.

---

## 🚀 Features

### 🧠 Smart Guessing Engine
- Probability-based player prediction
- Dynamic question selection
- Smart filtering algorithm
- Confidence score calculation

### 🤖 Gemini AI Integration
- Natural language questions
- Human-like interaction
- AI-generated explanations
- Conversational gameplay

### 🎯 Intelligent Question Flow
- Yes / No / Maybe / Don't Know support
- Adaptive questioning
- Best-question selection logic
- Real-time probability updates

### 📊 IPL Dataset
- 45+ IPL players
- Multiple teams support
- Batting & bowling styles
- Captaincy and activity tracking
